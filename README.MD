**Atividade_teste_um**

**Este projeto foi criado com o objetivo de registrar usuários, e cadastralos no banco de dados.
Aqui usamos linguagem de PHP para realizar cadastros, e também LogOuts de usuários. quando Um 
usrário loga no site, ele é redirecionado para uma pagina onde ele pode adicionar novos usuários ao 
banco de dados. 

** Também foi feito um processo de segurança onde caso o usuário não tenha Login ele será instantâniamente
redirecionado para a pagina de Login.
